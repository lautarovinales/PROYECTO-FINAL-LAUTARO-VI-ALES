Esto es un contenedor

"Contenedor" es el nombre que se le puede dar a un div que va a tener elementos dentro.

En este caso el nombre de este contenedor es "cont-1" al igual que el que está al lado, solamente que la diferencia esta en los contenidos de cada uno.

Algunas caracteristicas de este contenedor

1 - Este contenedor al igual que el que está al lado estan dentro de otro contenedor que se le conoce como "flexbox".

2 - Este contenedor cuenta con un "border-color":white y un "margin" de 1vw.

Esto tambien es un contenedor

PERO éste contenedor cuenta con otro contenido, este además de tener un título y un párrafo también cuenta con una imagen dentro de un contenedor.

Imagen: "Wombat" animal de origen Australiano.
Esto es un Aside

Un "Aside" se caracteriza por ir a un costado de un sitio web, en éste caso el lado derecho y de color violeta para que sea mas facil diferenciarlo.

Acá va el contenido principal o "main"

Este main tiene varios div que voy a ir separando por medio de colores para que sea facil de diferenciar. Por ejemplo, éste div de color verde agua, lleva solo unos parrafos y un titulo encima de todo.
